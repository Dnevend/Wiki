网站

保存使用过的网站, 和实用工具.

🧑‍💻 开发​

DevTooly All-in-one 前端开发工具导航. 聚合了前端开发有关的(CSS 动画、组件、样式、设计...)资源.

Web Dev Checklist Web 开发项目检查清单. 每一个检查项中提供了一些帮助信息和测试工具，非常有用.

Oh Shit Git 开发中常见的 Git 问题解决备忘清单.

JsonPlaceHolder 用于测试和原型设计中使用的 Mock API。

Quick Ref 为开发人员分享快速参考和备忘单.

Dev Urls 开发相关网站的聚合导航站点, 聚合了各主流开发社区的最近热门内容.

界面灵感 Web/App/Email/Others 灵感类网站合集.

Sass Pages 专注登陆页网站的集合.

sasspo SaaS 网站设计灵感集合.

Godly 来自互联网各地的天文数字般的网页设计灵感.

Landing Love 网站视觉动效.

Haikei SVG 设计资源生成.

Footer “The only footer gallery on earth.” 网站页脚(底导)设计.

Google Fonts 谷歌字体库.

AntDesign 设计模式

Framer Motion examples framer motion example projects.

🛠️ 工具​

Svg Wave 一款制作波浪图片或 SVG 动画的工具.

UI Verse UI 组件库，全部由 HTML & CSS 制作.

Good Palette UI 配色生成工具.

Hover.css CSS :hover 动画效果集合.

Animotion CSS 动画低代码工具.

Purple Photo “在几秒钟内创建漂亮的屏幕截图和帖子，无需任何设计技能” 截图美化、文章封面图制作工具.

Tiny PNG 图片压缩.

Record Screen 直接使用浏览器录屏.

kagi 输入网址, 就能快速总结出该网址文章的大意, 支持中文网址.

Email Helper 使用 OpenAI 自动生成商务邮件, 只要提供邮件主题即可, 支持中文.

Griddy 一款可以在线生成简单的 css grid 布局样式代码的 GUI 工具.

miniPaint 在线图像编辑器, Photoshop 替代品, 关键实现：图层、滤镜、HTML5、开源。
